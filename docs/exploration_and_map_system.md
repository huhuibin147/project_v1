# 探索与地图系统设计文档

## 概述

探索与地图系统定义游戏世界的空间结构，包含地图渲染、瓦片系统、交互物件、怪物刷新、天气昼夜、NPC日程和探索要素。

## 系统架构

```
┌──────────────────────────────────────────────────────┐
│                    探索与地图系统                      │
├────────────┬────────────┬────────────┬───────────────┤
│  地图系统   │  瓦片渲染   │  天气昼夜   │  探索要素     │
│  场景切换   │  多层绘制   │  视觉滤镜   │  宝箱采集     │
│  传送门     │  深度排序   │  粒子效果   │  隐藏通道     │
│  怪物刷新   │  建筑系统   │  战斗影响   │  NPC日程     │
├────────────┴────────────┴────────────┴───────────────┤
│  config/maps/ (地图数据)  config/tiles.json (瓦片)     │
│  frontend/js/map.js  tools/map_generator.py           │
└──────────────────────────────────────────────────────┘
```

## 地图数据格式

```json
{
  "id": "map_id",
  "name": "显示名称",
  "width": 50,
  "height": 40,
  "tile_size": 32,
  "layers": { "ground": [[tile_id, ...], ...] },
  "monsters": [{ "monster_id": "...", "x": N, "y": N, "patrol": [...] }],
  "objects": [{ "id": "...", "type": "...", "x": N, "y": N, "properties": {...} }],
  "npcs": [{ "npc_id": "...", "x": N, "y": N }],
  "player_spawn": { "x": N, "y": N }
}
```

## 交互物件

| 类型 | 说明 | 交互方式 |
|------|------|----------|
| portal | 传送门 | 踩上去自动触发 |
| chest | 宝箱 | 按 E 打开（一次性） |
| gather | 采集点 | 按 E 采集（有冷却） |
| decoration | 装饰物 | 按 E 查看文字 |

## 怪物刷新

地图 JSON 中的 `monsters` 数组定义怪物刷新点：

```json
{
  "monster_id": "wild_wolf",
  "x": 40, "y": 20,
  "patrol": [{"x": 38, "y": 20}, {"x": 42, "y": 20}, {"x": 42, "y": 24}]
}
```

- `patrol` 可选，有则怪物在路径点间巡逻，无则原地站立
- 村庄地图 `monsters: []`（安全区）

# 第二部分：瓦片与渲染系统

## 瓦片系统

瓦片定义在 `config/tiles.json`，每种瓦片有：名称、颜色、是否可行走、渲染细节标识。

当前 15 种瓦片：草地、泥土路、砖墙、屋顶、树木、水面、木地板、栅栏、石头、石板路、沙地、雪地、洞穴墙、洞穴地板等。

## 美术系统升级（规划）

### 多层瓦片渲染

- 地面层（基础瓦片）
- 装饰层（草丛、碎石、水波纹等细节）
- 阴影层（建筑和树木的投影）
- 天气层（雨雪粒子叠加）

### 建筑类型系统

建筑由多个瓦片组合构成，包含墙壁、屋顶、门、窗等元素。每种建筑类型有预设模板，支持自动生成和手动放置。

### 动画瓦片

- 水面波纹动画
- 火把闪烁动画
- 熔岩流动动画
- 传送门旋转动画

# 第三部分：天气与昼夜系统（规划）

## 昼夜系统

- 游戏内时间流速：现实 1 分钟 = 游戏 1 小时
- 4 个时段：

| 时段 | 游戏时间 | 视觉效果 | 玩法影响 |
|------|----------|----------|----------|
| 白天 | 6:00-18:00 | 明亮 | 无 |
| 黄昏 | 18:00-20:00 | 暖色滤镜 | 无 |
| 夜晚 | 20:00-6:00 | 暗蓝滤镜 | 怪物刷新 +20%，部分 NPC 休息 |
| 黎明 | 5:00-6:00 | 淡粉滤镜 | 无 |

## 天气系统

| 天气 | 视觉效果 | 玩法影响 |
|------|----------|----------|
| 晴天 | 无 | 无 |
| 阴天 | 轻微变暗 | 无 |
| 雨天 | 雨滴粒子 | 火系伤害 -10% |
| 雪天 | 雪花粒子 | 冰系伤害 +10% |
| 雷暴 | 闪电 + 雨滴 | 速度 -10% |

- 天气随机切换（每 10-30 分钟变化）
- 前端 Canvas 叠加色调滤镜和天气粒子
- 新增 API：`GET /api/world/status`（返回当前时间、天气）

# 第四部分：NPC 日程系统（规划）

## 设计目标

NPC 按时段切换位置和行为，增强世界真实感。

## 日程数据结构

```json
{
  "npc_id": "blacksmith",
  "schedule": [
    {"time_range": [6, 18], "location": "blacksmith_shop", "behavior": "work", "interactable": true},
    {"time_range": [18, 22], "location": "tavern", "behavior": "rest", "interactable": true},
    {"time_range": [22, 6], "location": "blacksmith_house", "behavior": "sleep", "interactable": false}
  ]
}
```

## 日程规则

- NPC 在指定时段出现在指定位置
- 非交互时段显示"已休息"提示
- 日程切换时 NPC 平滑移动到新位置（可选，或直接切换）

## 怪物生成工具

### 模板

| 模板名 | 类型 | HP倍率 | ATK倍率 | 经验倍率 | AI行为 |
|--------|------|--------|---------|---------|--------|
| normal | 普通 | ×1.0 | ×1.0 | ×1.0 | 攻击型 |
| elite | 精英 | ×2.0 | ×1.5 | ×2.5 | 攻击型 |

### 使用方法

```bash
cd tools
python monster_generator.py create-normal skeleton "骷髅兵" 2
python monster_generator.py create-elite dark_knight "黑暗骑士" 8
python monster_generator.py create-boss dragon "远古巨龙" 15
python monster_generator.py batch normal 5 "1-3" "森林"
python monster_generator.py apply
```

# 第七部分：地图生成工具

`tools/map_generator.py` — 创建、编辑、验证和预览游戏地图。

## 功能

1. 生成新地图（村庄、森林、洞穴、城镇、沙漠模板）
2. 格式化地图（清理JSON格式）
3. 扩展地图（扩大尺寸，保持原有内容居中）
4. 修复地图（移除多层边界墙，确保可通行）
5. 验证地图（有效性、可达性、对象位置）
6. 预览地图（终端字符形式）
7. 列出地图（显示基本信息）

## 使用方法

```bash
python tools/map_generator.py list
python tools/map_generator.py generate village my_village
python tools/map_generator.py preview village
python tools/map_generator.py validate village
python tools/map_generator.py expand village 60 50
python tools/map_generator.py fix village
```
