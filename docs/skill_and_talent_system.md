# 技能与天赋系统设计文档

## 概述

技能与天赋系统是角色成长的核心玩法。技能是玩家主动释放的战斗能力，天赋是被动加成和战斗触发效果。两者结合决定了角色的战斗风格和成长方向。

## 系统架构

```
┌─────────────────────────────────────────────────────────────────┐
│                        角色成长系统                              │
├──────────────────────┬──────────────────────────────────────────┤
│      技能系统         │              天赋系统                     │
│  ┌──────────────┐    │    ┌──────────────────────────────┐     │
│  │ 主动释放      │    │    │ 被动生效                      │     │
│  │ MP 消耗       │    │    │ 天赋点消耗                    │     │
│  │ 冷却回合      │    │    │ Lv.5 解锁                    │     │
│  │ 职业限制      │    │    │ 每 3 级获得 1 点              │     │
│  └──────┬───────┘    │    └──────────────┬───────────────┘     │
│         │            │                   │                      │
│         ▼            │                   ▼                      │
│  ┌──────────────┐    │    ┌──────────────────────────────┐     │
│  │ skills.json  │    │    │ talents.json                 │     │
│  │ skill_system │    │    │ talent_system                │     │
│  │ 11 个技能    │    │    │ 30 个天赋节点                 │     │
│  └──────────────┘    │    └──────────────────────────────┘     │
├──────────────────────┴──────────────────────────────────────────┤
│                        战斗引擎集成                              │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 技能释放 → 伤害计算 → 效果触发 → 天赋被动 → 战斗日志    │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
```

# 第一部分：技能系统

## 技能数据结构

```json
{
  "skill_id": "fireball",
  "name": "火球术",
  "description": "发射火球，造成 1.5 倍魔法伤害，30% 概率灼烧",
  "mp_cost": 15,
  "cooldown": 0,
  "type": "damage",
  "target": "enemy",
  "damage_type": "magic",
  "power": 1.5,
  "effects": [{"type": "burn", "duration": 3, "chance": 0.3}],
  "class_requirement": ["mage"],
  "level_requirement": 1
}
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `skill_id` | string | 唯一标识 |
| `name` | string | 显示名称 |
| `description` | string | 技能描述 |
| `mp_cost` | int | MP 消耗 |
| `cooldown` | int | 冷却回合数（0 表示无冷却） |
| `type` | string | 技能类型：`damage` / `heal` / `buff` / `debuff` |
| `target` | string | 目标：`enemy` / `self` / `ally` |
| `damage_type` | string | 伤害类型：`magic` / `physical` |
| `power` | float | 威力倍率（× 攻击力） |
| `effects` | list | 附加效果列表 |
| `class_requirement` | list | 职业限制（空数组表示通用） |
| `level_requirement` | int | 等级要求 |

## 技能类型

| 类型 | 说明 | 示例 |
|------|------|------|
| `damage` | 造成伤害，受 power 倍率影响 | 重击、火球术、背刺 |
| `heal` | 恢复生命值 | 治愈术 |
| `buff` | 增益自身，持续 N 回合 | 盾墙、狂暴、闪避 |
| `debuff` | 削弱敌人，持续 N 回合 | 毒刃（中毒效果） |

## 技能学习规则

- 每个职业有 3-5 个专属技能
- 学习前校验：职业匹配、等级要求、是否已学会
- 技能通过导师 NPC 学习（如导师艾尔文）

## 技能战斗流程

```
玩家选择技能
  │
  ├─ 检查 MP 是否充足
  ├─ 检查冷却是否就绪
  │
  ▼
消耗 MP，进入冷却
  │
  ▼
计算伤害/效果
  │
  ├─ damage 类型：基础伤害 × power × 浮动
  ├─ heal 类型：固定值 + 天赋加成
  └─ buff/debuff 类型：施加状态效果
  │
  ▼
触发附加效果（灼烧/冻结/中毒等）
  │
  ▼
天赋技能增强（skill_enhance）
  │
  ▼
记录战斗日志
```

## 技能伤害公式

```
基础伤害 = 攻击力 × power × (100 / (100 + 防御力))
浮动范围 = 基础伤害 × random(0.9, 1.1)
天赋增强 = 基础伤害 × skill_enhance_damage_boost
最终伤害 = max(1, floor(浮动范围 + 天赋增强))
```

## 群体技能系统

### 概述

群体技能（AOE）可同时影响多个目标，是多敌人战斗中的核心策略。当前后端已支持 `aoe: true` 标记和伤害递减机制，但 `skills.json` 中尚无群体技能。

### 群体技能类型

| 类型 | target | 说明 | 示例 |
|------|--------|------|------|
| 群体伤害 | `enemy` + `aoe: true` | 对所有存活敌人造成伤害，按目标数递减 | 旋风斩、烈焰风暴 |
| 群体治疗 | `self` + `aoe: true` | 恢复自身大量生命（无递减） | 神圣祈祷 |
| 群体增益 | `self` + `aoe: true` | 为自身施加强力增益 | 战吼 |
| 群体 debuff | `enemy` + `aoe: true` | 对所有敌人施加 debuff | 恐惧嚎叫 |

### AOE 统一伤害乘数

AOE 技能对所有目标造成**统一比例的伤害**，不再随目标数量变化。该比例低于单体技能，以平衡其群体攻击的优势。

```
统一乘数: 65%
```

即：AOE 技能的最终伤害 = 技能威力 × 攻击力 × 0.65（对所有目标一致）

附加效果（灼烧/中毒等）对每个目标独立判定概率。

### 前端 AOE 技能 UI 适配

1. **技能面板**：AOE 技能显示 `群体` 标签（已有）
2. **目标选择**：AOE 技能无需选择目标，直接使用（需跳过目标选择步骤）
3. **伤害数字**：AOE 技能对每个存活怪物分别显示伤害数字（需改进：按 targets 数据分别显示）
4. **技能描述**：描述中标注"对所有敌人"或"群体"
5. **视觉效果**：AOE 技能释放时所有怪物同时受击闪烁

### 怪物群体技能（扩展）

BOSS 怪物可配置群体特殊技能：

```json
{
  "special": {
    "type": "aoe_attack",
    "damage_multiplier": 0.8,
    "effect": "burn",
    "chance": 0.3,
    "duration": 2,
    "message": "暗影树精释放暗焰风暴！"
  }
}
```

怪物 AI 的 `special` 类型扩展：
- `apply_effect`：单体施加效果（现有）
- `aoe_attack`：群体攻击 + 可选效果（新增）
- `heal`：自我治疗（新增）
- `summon`：召唤小怪（未来扩展）

# 第二部分：天赋系统

## 概述

天赋系统在玩家达到 Lv.5 时解锁，提供被动加成和战斗触发效果，决定角色的成长方向。每个职业有 2 条专精路线，每条路线 5 个天赋节点，共 30 个天赋。

## 核心规则

### 天赋点获取

```
解锁等级 = 5
天赋点间隔 = 3 级
天赋点数量 = (等级 - 5) // 3 + 1（等级 ≥ 5 时）
```

| 等级 | 天赋点 | 说明 |
|------|--------|------|
| 1-4 | 0 | 未解锁 |
| 5 | 1 | 解锁天赋系统 |
| 6-7 | 1 | 等待升级 |
| 8 | 2 | 获得第 2 点 |
| 11 | 3 | 获得第 3 点 |
| 14 | 4 | 获得第 4 点 |
| 17 | 5 | 获得第 5 点 |
| 20 | 6 | 获得第 6 点 |

### 学习规则

- 每条路线需按层级顺序解锁（tier 1 → 2 → 3 → 4 → 5）
- 两条路线可混合点，但高层天赋需要同路线前置
- 学习前校验：职业匹配、等级要求、前置天赋、天赋点充足
- 学习后立即重新计算角色属性

### 重置规则

```
重置费用 = 100 + 等级 × 50
```

| 等级 | 重置费用 |
|------|----------|
| 5 | 350 |
| 10 | 600 |
| 15 | 850 |
| 20 | 1100 |

## 天赋效果类型详解

### `stat_boost` — 属性加成

直接修改角色属性，支持百分比加成。

```json
{"type": "stat_boost", "stat": "attack", "value": 0.08, "is_percent": true}
```

支持的属性：`attack`、`defense`、`speed`、`max_hp`、`max_mp`、`crit_chance`、`crit_damage`、`dodge_chance`

计算方式：
```
总加成 = Σ 所有 stat_boost 效果
最终属性 = 基础属性 × (1 + 总加成)
```

### `on_kill` — 击杀触发

击杀怪物时触发效果。

```json
{"type": "on_kill", "action": "heal", "value": 10}
```

支持的动作：
- `heal` — 恢复 HP（value 为百分比）
- `attack_boost` — 临时提升攻击力

### `on_attack` — 攻击触发

普通攻击时触发效果。

```json
{"type": "on_attack", "action": "lifesteal", "value": 15}
{"type": "on_attack", "action": "chance_bleed", "chance": 0.2, "duration": 3}
{"type": "on_attack", "action": "chance_stun", "chance": 0.15, "duration": 1}
```

支持的动作：
- `lifesteal` — 吸血（value 为伤害转化百分比）
- `chance_bleed` — 概率施加流血
- `chance_stun` — 概率施加眩晕

### `on_combat_start` — 战斗开始触发

进入战斗时触发效果。

```json
{"type": "on_combat_start", "action": "chance_stun", "chance": 0.25, "duration": 1}
```

### `on_dodge` — 闪避触发

成功闪避攻击时触发效果。

```json
{"type": "on_dodge", "action": "counter_attack", "value": 50}
```

### `conditional` — 条件触发

每回合检查条件，满足时触发。

```json
{"type": "conditional", "condition": "hp_below", "threshold": 0.3, "action": "attack_boost", "value": 25}
```

支持的条件：
- `hp_below` — HP 低于阈值（threshold 为百分比）
- `mp_above` — MP 高于阈值

### `skill_enhance` — 技能强化

强化特定技能的效果，与技能系统联动。

```json
{"type": "skill_enhance", "skill_id": "fireball", "enhance": {"damage_boost": 0.2, "burn_chance_boost": 0.2}}
```

### `passive` — 常驻被动

持续生效的被动效果。

```json
{"type": "passive", "action": "defend_boost", "value": 0.75}
{"type": "passive", "action": "last_stand"}
{"type": "passive", "action": "mp_regen", "value": 0.03}
{"type": "passive", "action": "element_combo", "value": 0.3}
{"type": "passive", "action": "element_storm", "damage": 0.5, "interval": 3}
```

支持的动作：
- `defend_boost` — 提升防御姿态减伤比例
- `last_stand` — 不屈意志（致命伤害保留 1 HP）
- `mp_regen` — 每回合自动回复 MP（value 为百分比）
- `element_combo` — 元素连击加成
- `element_storm` — 元素风暴（自动群体伤害）

## 天赋战斗集成

天赋被动通过 `get_talent_passives()` 传入战斗快照，在 `CombatSession` 中自动触发：

```
战斗开始
  │
  ├─ _apply_talent_on_combat_start()
  │   └─ 烟雾弹等战斗开始效果
  │
每回合开始
  │
  ├─ _apply_talent_conditional()
  │   └─ 检查 HP/MP 条件，触发狂暴之心等
  │
  ├─ _apply_talent_mp_regen()
  │   └─ 魔力涌流回复 MP
  │
  ├─ _apply_talent_element_storm()
  │   └─ 元素风暴自动释放
  │
玩家攻击
  │
  ├─ _apply_talent_on_attack()
  │   ├─ 吸血（恢复 HP）
  │   ├─ 概率流血
  │   └─ 概率眩晕
  │
  └─ _apply_talent_skill_enhance()
      └─ 技能强化（伤害加成/效果增强）
  │
击杀怪物
  │
  ├─ _apply_talent_on_kill()
  │   ├─ 恢复 HP（嗜血本能）
  │   └─ 攻击力提升（战神附体）
  │
玩家闪避
  │
  └─ _apply_talent_on_dodge()
      └─ 影分身反击
  │
玩家防御
  │
  └─ _apply_talent_defend_boost()
      └─ 不屈意志（提升减伤）
  │
受到致命伤害
  │
  └─ _apply_talent_last_stand()
      └─ 坚韧（保留 1 HP）
```

## 数据结构

```json
{
```

### 天赋字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `talent_id` | string | 唯一标识，格式：`职业_路线_天赋名` |
| `name` | string | 显示名称 |
| `class` | string | 适用职业：`warrior`/`rogue`/`mage` |
| `tree` | string | 天赋树名称 |
| `tier` | int | 层级（1-5） |
| `description` | string | 效果描述 |
| `effects` | list | 效果列表 |
| `prerequisites` | list | 前置天赋 ID 列表 |

### 被动效果结构

```python
{
    "on_kill": [],              # 击杀触发效果列表
    "on_attack": [],            # 攻击触发效果列表
    "on_combat_start": [],      # 战斗开始触发效果列表
    "on_dodge": [],             # 闪避触发效果列表
    "conditional": [],          # 条件触发效果列表
    "skill_enhance": {},        # 技能强化字典 {skill_id: [enhance]}
    "defend_boost": 0.5,        # 防御减伤比例
    "last_stand": False,        # 不屈意志标志
    "mp_regen": 0.0,            # MP 回复百分比
    "element_combo": 0.0,       # 元素连击加成

}
```

## 扩展方向

- **技能升级**：已学技能可升级，提升威力或降低消耗
- **天赋共鸣**：同时点亮两条路线的特定天赋时触发额外效果
- **天赋装备**：通过装备获得额外天赋点或特殊天赋
- **天赋重置道具**：免费重置道具作为稀有掉落
- **天赋等级**：已学天赋可升级，提升效果数值
- **跨职业天赋**：特定条件解锁其他职业的天赋
- **技能连招**：连续使用特定技能组合触发额外效果
