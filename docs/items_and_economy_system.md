# 物品与经济系统设计文档

## 概述

物品与经济系统提供游戏的核心循环：战斗→材料→锻造→更强装备。包含背包管理、装备穿戴、消耗品使用、锻造升级、词条随机化、商店交易和掉落系统。

## 系统架构

```
┌──────────────────────────────────────────────────────┐
│                    物品与经济系统                      │
├────────────┬────────────┬────────────┬───────────────┤
│  背包系统   │  装备系统   │  锻造系统   │  经济系统     │
│  物品管理   │  穿戴加成   │  配方锻造   │  商店交易     │
│  堆叠排序   │  稀有度     │  成功率     │  掉落系统     │
│  使用消耗   │  词条系统   │  材料收集   │  金币流通     │
├────────────┴────────────┴────────────┴───────────────┤
│              config/items.json (60+ 物品)              │
│              backend/item_system.py                    │
│              tools/item_generator.py                   │
└──────────────────────────────────────────────────────┘
```

# 第一部分：背包系统

## 物品类型

| 类型 | 说明 | 堆叠上限 |
|------|------|----------|
| weapon | 武器，可装备 | 1 |
| armor | 防具，可装备 | 1 |
| accessory | 饰品，可装备 | 1 |
| consumable | 消耗品（药水、卷轴） | 99 |
| food | 食物，恢复 HP | 99 |
| tool | 工具 | 10 |
| material | 材料 | 99 |
| skill_book | 技能书，学习技能 | 1 |

## 背包规则

- 全局背包，无格数限制
- 物品可堆叠，堆叠上限因类型不同
- 支持按类型排序显示
- 消耗品和食物可在背包中直接使用（非战斗状态）
- 战斗中可通过「物品」按钮使用消耗品
- 技能书在背包中使用，学会对应技能

## 背包界面

支持两种展示模式和标签筛选。

### 视图切换

- **列表视图**（默认）：纵向排列，显示完整信息（名称、类型、属性、描述、操作按钮）
- **格子视图**：4列网格，显示图标、名称、数量角标，点击弹出详情提示框

### 标签筛选

| 标签 | 包含类型 |
|------|---------|
| 全部 | 所有物品 |
| 武器 | `weapon` 类型或 `equip_slot: weapon` |
| 防具 | `armor` 类型或装备槽为 shield/head/body |
| 消耗 | `consumable` 类型（药水、技能书等） |
| 材料 | `material` 类型 |

# 第二部分：装备系统

## 装备槽位

| 槽位 | key | 可装备类型 |
|------|-----|-----------|
| 武器 | `weapon` | weapon |
| 盾牌 | `shield` | armor（shield 子类） |
| 头部 | `head` | armor（head 子类） |
| 身体 | `body` | armor（body 子类） |
| 饰品 | `accessory` | accessory |

## 属性计算

```
实际属性 = 职业基础 + 等级成长 + 装备加成总和 + 天赋加成
```

装备属性支持：攻击(attack)、防御(defense)、速度(speed)、最大生命值(max_hp)、最大魔法值(max_mp)

## 稀有度

| 稀有度 | key | 颜色 | 价格倍率 | 词条数 |
|--------|-----|------|----------|--------|
| 普通 | `common` | 灰 #aaaaaa | ×1.0 | 0 |
| 优秀 | `uncommon` | 绿 #1eff00 | ×1.3 | 1 |
| 稀有 | `rare` | 蓝 #0070dd | ×1.8 | 2 |
| 史诗 | `epic` | 紫 #a335ee | ×2.5 | 3 |
| 传说 | `legendary` | 橙 #ff8000 | ×4.0 | 4 |

## 等级段

| 等级段 | key | 使用上限 | 属性倍率 | 价格倍率 |
|--------|-----|---------|---------|---------|
| 初级 | `tier1` | Lv.4 | ×1.0 | ×1.0 |
| 中级 | `tier2` | Lv.9 | ×1.5 | ×1.5 |
| 高级 | `tier3` | Lv.15 | ×2.2 | ×2.5 |

## 职业限定

装备可设定 `class_requirement` 字段，限制只有特定职业才能装备。空数组 `[]` 表示不限职业。

## 装备对比

背包和商店中的可装备物品显示与当前装备的属性差异：绿色 `+N` 表示提升，红色 `-N` 表示下降。

# 第四部分：锻造系统（规划）

## 设计目标

玩家收集材料后在铁匠铺锻造高级装备，形成「战斗→材料→锻造→更强装备」的核心循环。

## 锻造规则

- 锻造需要配方 + 材料 + 金币
- 锻造有成功率，失败返还部分材料
- 高级锻造需要高等级（等级不足无法锻造）
- 锻造产出的装备可附带随机词条

## 锻造配方数据结构

```json
{
  "recipe_id": "forge_iron_sword",
  "name": "锻造铁剑",
  "output": {"item_id": "iron_sword", "quantity": 1},
  "materials": [
    {"item_id": "iron_ore", "quantity": 3},
    {"item_id": "wood", "quantity": 1}
  ],
  "gold_cost": 50,
  "level_requirement": 3,
  "success_rate": 1.0,
  "category": "weapon",
  "tier": "basic"
}
```

## 锻造等级段

| 等级段 | 等级要求 | 成功率基础 | 说明 |
|--------|----------|------------|------|
| 初级 | Lv.1 | 100% | 铁制装备 |
| 中级 | Lv.5 | 90% | 秘银装备 |
| 高级 | Lv.10 | 75% | 精金装备 |
| 大师 | Lv.15 | 60% | 传说装备 |

# 第五部分：词条系统（规划）

## 设计目标

装备可附带随机词条，增加装备多样性和收集驱动力。

## 词条分类

| 类别 | 触发条件 | 示例 |
|------|----------|------|
| 攻击触发 | 造成伤害时 | 灼烧、冰冻、雷击 |
| 被动加成 | 始终生效 | 暴击率+、闪避率+、攻击力+% |
| 受击触发 | 受到伤害时 | 荆棘（反伤）、护盾（概率减伤） |
| 击杀触发 | 击杀怪物时 | 吸血、经验加成 |
| 条件触发 | 特定条件 | 低血量攻击+、MP低于30%防御+ |

## 词条数据结构

```json
{
  "affix_id": "burning",
  "name": "灼烧",
  "category": "on_attack",
  "description": "攻击时 15% 概率附加灼烧效果",
  "effects": [
    {"type": "on_hit", "trigger_chance": 0.15, "apply_effect": "burn", "effect_duration": 3}
  ],
  "tier": 2,
  "weight": 30,
  "level_range": {"min": 1, "max": 15},
  "compatible_slots": ["weapon"]
}
```

## 词条列表（规划）

| 词条 | 类别 | 效果 | 适用槽位 |
|------|------|------|----------|
| 锋利 | 被动 | 攻击力 +5% | 武器 |
| 灼烧 | 攻击触发 | 15% 概率附加灼烧 | 武器 |
| 冰霜 | 攻击触发 | 10% 概率冻结 1 回合 | 武器 |
| 雷击 | 攻击触发 | 8% 概率额外 50% 伤害 | 武器 |
| 吸血 | 攻击触发 | 造成伤害的 10% 恢复 HP | 武器 |
| 坚固 | 被动 | 防御力 +8% | 防具 |
| 荆棘 | 受击触发 | 受击时反弹 15% 伤害 | 防具 |
| 护盾 | 受击触发 | 10% 概率减伤 50% | 防具 |
| 灵活 | 被动 | 速度 +5 | 饰品 |
| 暴击 | 被动 | 暴击率 +5% | 武器/饰品 |
| 生命 | 被动 | 最大 HP +10% | 防具/饰品 |
| 魔力 | 被动 | 最大 MP +10% | 饰品 |
| 贪婪 | 击杀触发 | 金币获取 +15% | 饰品 |
| 猎手 | 击杀触发 | 经验获取 +10% | 饰品 |
| 绝境 | 条件触发 | HP<30% 时攻击 +20% | 武器 |

## 词条生成规则

- 装备稀有度决定词条数量（普通 0、优秀 1、稀有 2、史诗 3、传说 4）
- 词条从对应等级段的词条池中按权重随机抽取
- 同一件装备不重复同种词条
- 锻造产出的装备自动附带随机词条
- 附魔 NPC 可为已有装备添加/替换词条（花费金币和材料）

# 第六部分：经济系统

## 货币

- 单一货币：金币
- 来源：怪物掉落、任务奖励、出售物品
- 消耗：购买物品、锻造、附魔、治疗服务、天赋重置

## 商店系统

- NPC 商店有固定库存，出售价格 = 物品基础价格
- 玩家出售价格 = 物品基础价格 × 0.5
- NPC 收购的物品存入独立的收购仓库，不混入商店库存

### 交易方式

1. **对话驱动**：玩家在对话中表达交易意图，LLM 识别后自动执行
2. **商店面板**：按 E 打开 NPC 交互选项，选择"商店"直接交易

### 交易校验

- 购买：金币充足 + NPC 库存足够 + 物品可买
- 出售：玩家持有物品 + 物品可卖 + NPC 金币充足

## 掉落系统

- 怪物掉落独立概率判定（每项物品单独 roll）
- 掉落受等级差影响：玩家等级高于怪物 5 级以上时，掉落率降低
- BOSS 掉落包含唯一物品（100% 概率）和稀有物品（10-30% 概率）

### 掉落物品规则

| 种族 | 材料掉落 | 消耗品掉落 | 装备掉落 | 基础概率 |
|------|---------|-----------|---------|---------|
| undead | 兽骨、魔晶、布料 | 生命药水、净化药水 | 武器、饰品 | 40% |
| beast | 狼皮、兽骨、毒液囊 | 生命药水、解毒药 | 防具、饰品 | 50% |
| humanoid | 铁矿、布料、钢锭 | 生命药水、绷带、魔力药水 | 武器、防具、盾牌 | 35% |
| dragon | 龙鳞、魔晶、晶片 | 强效生命药水、力量药剂 | 武器、防具、饰品 | 70% |
| default | 草药、蘑菇、布料 | 生命药水、绷带 | 无 | 30% |

- 精英怪物：装备掉落概率提升，可能掉落稀有装备
- BOSS：额外掉落史诗/传说装备，概率 10%

## NPC 商店分配

| NPC | 商店名 | 主营 |
|-----|--------|------|
| 铁匠老王 | 老王铁匠铺 | 武器、盾牌、头部、身体、药水、工具、战士技能书 |
| 杂货婆刘婶 | 刘婶杂货铺 | 食物、身体（布袍）、日用品、法师技能书 |
| 采药人老林 | 老林药摊 | 药水、饰品、头部、草药、盗贼技能书 |

# 第八部分：物品生成工具

`tools/item_generator.py` — 用于批量生成游戏物品和 NPC 商店库存。

## 使用方法

```bash
cd tools
python item_generator.py generate equip tier1       # 初级装备
python item_generator.py generate equip all         # 所有等级段装备
python item_generator.py generate consumable        # 消耗品
python item_generator.py generate all               # 所有类型
python item_generator.py apply                      # 应用到 items.json
python item_generator.py shop all                   # 更新NPC商店库存
python item_generator.py validate                   # 验证数据
python item_generator.py preview                    # 属性分布预览
```

## 属性计算公式

```
总预算 = 等级中值 × 1.5
单项属性 = 总预算 × 权重（max_hp 额外 ×3）
价格 = (攻击×10 + 防御×8 + 速度×6 + HP×2 - 负面属性×3) × 等级倍率
```
