## 三、设计方案

### 3.1 商店与背包合并为双栏面板

**当前**：商店面板和背包面板独立，买卖时需切换。

**优化后**：商店打开时显示双栏面板，左侧为商店库存，右侧为玩家背包。

```
┌─────────────────────────────────────────────────────────────┐
│  老王铁匠铺                              你的金币: 500      │
│  商店金币: 300                            [×]               │
├──────────────────────────┬──────────────────────────────────┤
│  🏪 商店                 │  🎒 背包                         │
│  [搜索...] [全部▼] [排序▼]│  [搜索...] [全部▼] [排序▼]       │
│  ┌─────────────────────┐ │ ┌──────────────────────────────┐ │
│  │ ⚔️ 铁剑    售价:80  │ │ │ ⚔️ 铜剑    出售价:25        │ │
│  │ 攻+8  [购买-][1][+] │ │ │ 攻+5  [出售]                │ │
│  ├─────────────────────┤ │ ├──────────────────────────────┤ │
│  │ 🛡️ 铁盾   售价:60  │ │ │ 🧪 生命药水  出售价:5       │ │
│  │ 防+6  [购买-][1][+] │ │ │ 回复30HP  [出售] x3         │ │
│  └─────────────────────┘ │ └──────────────────────────────┘ │
├──────────────────────────┴──────────────────────────────────┤
│  交易消息区域                                                │
└─────────────────────────────────────────────────────────────┘
```

**关键设计**：

1. 面板宽度从 480px 扩展到 920px
2. 左栏为商店（460px），右栏为背包（460px）
3. 每栏独立搜索、筛选、排序
4. 购买操作在商店侧，出售操作在背包侧
5. 纯背包模式（I键打开）保持原宽度 560px

### 3.2 购买/出售数量选择器

**当前**：`doTrade` 固定 `quantity=1`，每次只能买/卖1个。

**优化后**：物品卡片内嵌数量选择器。

```html
<div class="qty-selector">
  <button class="qty-btn qty-minus" onclick="adjustQty('buy_iron_sword', -1)">−</button>
  <input class="qty-input" type="number" value="1" min="1" max="5" id="buy_iron_sword">
  <button class="qty-btn qty-plus" onclick="adjustQty('buy_iron_sword', 1)">+</button>
  <button class="btn-buy" onclick="doTrade('buy', 'iron_sword', getQty('buy_iron_sword'))">购买</button>
</div>
```

**规则**：
- 购买：max = 商店库存量，受限于玩家金币
- 出售：max = 玩家持有量
- Shift+点击按钮：直接设为最大数量
- 消耗品默认 max=持有量，装备默认 max=1

### 3.3 搜索/筛选/排序

**搜索**：实时模糊搜索（名称、描述、词条）

**筛选标签**：

| 标签 | 筛选条件 |
|------|----------|
| 全部 | 无筛选 |
| 武器 | type=weapon |
| 防具 | type=armor 或 equip_slot∈{shield,head,body} |
| 饰品 | equip_slot=accessory |
| 消耗 | type=consumable 或 type=food |
| 材料 | type=material |
| 可装备 | equip_slot 存在且未装备 |

**排序选项**：

| 排序 | 规则 |
|------|------|
| 默认 | 按配置文件顺序 |
| 稀有度 | legendary > epic > rare > uncommon > common |
| 价格↑ | buy_price/sell_price 升序 |
| 价格↓ | buy_price/sell_price 降序 |
| 名称 | 按名称拼音排序 |

### 3.4 快捷操作

| 操作 | 效果 |
|------|------|
| 双击装备 | 直接装备（背包内） |
| 双击消耗品 | 直接使用 |
| Shift+点击商店物品 | 快速购买1个 |
| Shift+点击背包物品（商店打开时） | 快速出售1个 |
| 右键物品 | 弹出上下文菜单（装备/使用/出售/详情） |

### 3.5 面板宽度优化

| 模式 | 宽度 | 列数 |
|------|------|------|
| 纯背包（I键） | 560px | 列表/4格 |
| 商店双栏 | 920px | 左右各460px |
| 格子视图 | 同上 | 列表/5格 |

### 3.6 Tooltip 增强

- 词条完整描述（不再截断）
- 装备来源提示（"商店购买"、"怪物掉落"、"锻造产出"）
- 属性条可视化对比（当前装备 vs 新装备，用颜色条表示）
